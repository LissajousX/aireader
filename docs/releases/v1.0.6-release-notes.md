## ⚠️ macOS 用户须知 / macOS Notice

当前 macOS 版本 **尚未进行 Apple 开发者签名和公证**（Apple Notarization），首次打开时 macOS 会阻止运行。请按以下方式操作：

The macOS build is **not yet notarized** with Apple. macOS will block it on first launch. Use one of these methods to open it:

### 方法一：右键打开（推荐）

1. 将 `Aireader.app` 拖入「应用程序」文件夹
2. **右键点击**（或 Control + 点击）`Aireader.app`
3. 选择「打开」
4. 在弹出的对话框中点击「打开」
5. 后续启动可正常双击打开

### 方法二：终端命令解除限制

```bash
# 解除隔离属性
xattr -cr /Applications/Aireader.app

# 然后正常双击打开即可
```

### 方法三：系统设置

1. 双击尝试打开 Aireader（会被阻止）
2. 打开「系统设置」→「隐私与安全性」
3. 在底部找到 "Aireader 已被阻止" → 点击「仍要打开」

> 后续版本将完成 Apple 签名和公证，届时可直接双击安装运行。

## ✨ 更新内容 / What's New (since v1.0.0)

### 🖥️ 跨平台
- 新增 macOS 支持（ARM + Intel），Metal GPU 自动启用
- 新增 Linux 支持（AppImage + Deb），Vulkan GPU 加速
- 修复 macOS 上 EPUB 资源路径和 WKWebView 选词问题

### 🤖 AI 增强
- Ollama 深度思考模式优化（真正可关闭）
- 新增模型采样参数配置（temperature、top_p 等）
- 修复 Windows CUDA 性能问题（DLL 加载路径修正）
- 改进硬件探测和 GPU 判断逻辑

### 🎨 界面优化
- 欢迎页功能卡片全新渐变配色
- 设置页内置 AI 区块间距优化
- 高级配置按钮样式更醒目
- 错误提示增加关闭按钮
- 彩色图标替代单色图标

### 🛡️ 稳定性
- 批量导入文件增加错误处理和校验
- 全面的错误处理增强，防止静默失败
- 新增 256+ 单元测试覆盖核心模块
- GitHub Actions CI/CD 自动化构建和发布
